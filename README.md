# my-first-game
My Scratch game for CS50 Week 0
![My Game Demo](ezgif-2c28e08e0bac88b8.gif)
## About the Game
This is my first project for CS50! I built this game using Scratch to learn the basics of computer science.

## How to Play 
there's a diver whom you have to save from sharks.if he touches any of the sharks or edges,the game is over.Just press space key to move him upwards else do nothing for downward motion
## What I Learned
* How to use **loops** to keep the game running.
* How to use **if-statements** to detect collisions.
*use of conditionals, variables and to make our own custom blocks
